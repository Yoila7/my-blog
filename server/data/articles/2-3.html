JavaScript是可插入HTML页面的编程代码

用法

内部

在<head>或<body>中插入<script>标签，直接在标签中写JS代码，标签位置会影响页面加载顺序。

外部

同样插入<script>标签，但是不直接写内容，而是用外部文件导入。

例如

<script src="myScript.js"></script>

输出方式

window.alert();

弹出警示框显示括号中的数据，可以是数字、表达式、字符串等。

操作HTML元素

document.getElementById(id)

利用id指定元素

.innerHTML

获取或修改元素内容（像一个变量）

例如

document.getElementById("demo").innerHTML = "段落已修改。";

写入HTML文档

document.write();

直接写入括号中的内容，常用于页面的调试

注意：页面加载中使用会插入该内容，页面加载完成后使用会覆盖整个页面。

写入控制台

console.log();

向调试控制台写入括号中的内容，并不影响原来页面的显示。

JS语法

JS字面量

数字：整数、小数或科学计数（e）

字符串：用单或双引号

表达式

数组：中括号 例如：[ 1, 2, 3 ]

对象：大括号和冒号 例如：{ firstname: "John", lastname: "Doe" }

函数：function 例如：function myFunction(a, b) { return a * b;}

JS变量

= 赋值

var：ES5 引入的变量声明方式，具有函数作用域。

let：ES6 引入的变量声明方式，具有块级作用域（即只在距离最近的一对大括号内有效）。

const：ES6 引入的常量声明方式，具有块级作用域，且值不可变。

JS注释

双斜杠// 或 /* */ 多行注释

JS语句

;分隔语句 {}组合语句 \折行

JS数据类型

变量均拥有动态类型

typeof 可用于查询变量类型

声明变量类型：var x=new Number/Boolean/String/Array/Object;

对象寻址方式：name=person.lastname;/name=person["lastname"];二者均可

Undefined and Null

变量为undefined表示未赋值或不需要，变量被赋值为null则会被释放内存

JS对象

本质上是键值对的集合

值也可以是函数

例如


var person = {
    firstName: "John",
    lastName : "Doe",
    fullName : function() {
        return this.firstName + " " + this.lastName;
    }
};
document.getElementById("demo").innerHTML = person.fullName();

JS作用域

函数外定义为全局变量，函数内定义为局部变量，未定义但是在函数内调用为全局变量

所有全局变量都属于window对象

JS事件

<some-HTML-element some-event="JavaScript 代码">

可以在发生某种事件时执行对应JS代码

常见事件

onchange HTML 元素改变
onclick 用户点击 HTML 元素
onmouseover 鼠标指针移动到指定的元素上时发生
onmouseout 用户从一个 HTML 元素上移开鼠标时发生
onkeydown 用户按下键盘按键
onload 浏览器已完成页面的加载

JS字符串

字符串长度

内置属性length可用于查询

例如：str.length

字符转义


\'	单引号
\"	双引号
\\	反斜杠
\n	换行
\r	回车
\t	tab(制表符)
\b	退格符
\f	换页符

模板字符串

用反引号`包裹

其中可以直接加入HTML代码，可以用${}包裹后加入表达式，表达式中可调用函数。

==和===

==为等于（值相同） 10=="10"为true

===为绝对等于（类型和值均相同） 10==="10"为false

JS for循环

两种方式

for (var i=0; i<5; i++) for (x in person)

JS标签

label:statements

以这种方式可以为语句或语句块起名为label


break label；跳出label语句
continue label；结束label语句的单次迭代

这两者可以用于在内部循环中跳出外部循环
